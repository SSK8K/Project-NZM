## Brief Description
This project is about the Theory of Languages and Automata course.  
It is a program that understands whether a given grammar of a language is regular or not. Then, if it is regular, it will create and show the DFA of the language graphically.  
The backend and frontend were made using C# with .NET 10. We then used Windows Forms to implement the frontend.
